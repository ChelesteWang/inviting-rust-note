## 编译过程

Code -> Tokens -> AST -> HIR -> MIR -> LLVM IR -> 机器码

AST->HIR: 类型检查，方法查找
HIR->MIR: 借用检查，优化，代码生成

Rustc 更详细的文档：  [Overview of the Compiler](https://rustc-dev-guide.rust-lang.org/overview.html)
